# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue on the
[GitHub repository](https://github.com/GIS-Inscribed-Geometry/IGE/issues)
or by contacting the maintainers directly.

We aim to acknowledge receipt of vulnerability reports within 48 hours
and will strive to release a fix as soon as possible.

Please do not disclose security vulnerabilities publicly until a fix has
been released.
